## Overview

**Studex** is a student-first marketplace built for the Virginia Tech community. It supports:
- Secure user authentication (email/password)
- Listings with images, categories, and pricing
- Real-time data storage and retrieval with Supabase
- Clean, fast UI powered by **React + Vite** and **Tailwind CSS**

---

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Backend:** Supabase (User Auth, Data Storage, Row-Level Security)

---

## Features

- **Auth:** Sign up, sign in, sign out, session persistence
- **Listings:** Create / edit / delete listings (permissioned)
- **Search & Filters:** Query listings by keyword, category, price, and recency
- **Image Uploads:** Supabase Storage for listing images
- **User Profiles:** Basic profile metadata and seller info
